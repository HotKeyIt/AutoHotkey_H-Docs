What is AutoHotkey Module

AutoHotkey Module is thought to be used in other programs.
AutoHotkey Module is running in its own context, using separate memory, functions and variables. You can access variables and objects from another thread.
AutoHotkey Module exports several functions that allow various operations to create and manipulate the new thread.

Use AutoHotkey in other programs: AutoHotkey Module can be used in other programming languages like VB, C#, C++, Python, Lua and many more. Programs that do not support loading a dll naturally can use the COM Interface.
