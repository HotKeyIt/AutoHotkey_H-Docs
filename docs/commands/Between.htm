Between

Checks whether a variable's contents are numerically or alphabetically between two values (inclusive).

Between, OutPutVar, VariableOrValue, LowerBound, UpperBound

Command  Example: Between, result,% var, 10, 100
Function Example: Between(var, 10, 100)

Parameters

OutPutVar

The name of the variable in which to store true/1 if variable is within the range or false/0 otherwise.

VariableOrValue

The variable name whose contents will be checked.

LowerBound

To be within the specified range, Var must be greater than or equal to this string or number.

UpperBound

To be within the specified range, Var must be less than or equal to this string or number.

Example

if between(var, 1, 5)
    MsgBox, %var% is in the range 1 to 5, inclusive.

if !between(var, 0.0, 1.0)
    MsgBox %var% is not in the range 0.0 to 1.0, inclusive.

if between(var, VarLow, VarHigh)
    MsgBox %var% is between %VarLow% and %VarHigh%.

if between(var, "blue", "red")
    MsgBox %var% is alphabetically between the words blue and red.

LowerLimit := 1
UpperLimit := 10
InputBox, UserInput, Enter a number between %LowerLimit% and %UpperLimit%
if !between(UserInput, LowerLimit, UpperLimit)
    MsgBox Your input is not within the valid range.
