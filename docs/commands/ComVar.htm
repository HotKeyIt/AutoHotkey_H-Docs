ComVar

Creates an object which can be used to pass a value ByRef.

ComVar, OutPutVar [, Type]

Command  Example: ComVar, var
Function Example: var := ComVar()

Parameters

OutPutVar

The name of the variable in which to store the ComVar object.

Type

Variant type, see ComObjType.

General Remarks

Var[]        ; retrieve the value
Var[] := Val ; set the value
Var.ref      ; retrieve a ByRef object for passing to a COM function.

Related

ComObjType, ComObjValue, ComObject, ComObjCreate, ComObjGet, ComObjActive

Examples

; Example: Pass a VARIANT ByRef to a COM function.

code := "
(
Sub Example(Var)
    MsgBox Var
    Var = "out value!"
End Sub
)"
sc := ComObjCreate("ScriptControl"), sc.Language := "VBScript", sc.AddCode(code)

var := ComVar()
var[] := "in value"
sc.Run("Example", var.ref)
MsgBox % var[]
