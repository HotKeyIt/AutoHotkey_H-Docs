ComVar

Creates an object which can be used to pass a value ByRef.

Parameters

ComVar, OutPutVar, Type
Example  Command: ComVar, var
Example Function: var := ComVar()
Type

An integer indicating the type of value. See ComObjType() for a list of types.

General Remarks

Var[] retrieves the value.
Var[] := Val sets the value.
Var.ref retrieves a ByRef object for passing to a COM function.

Related

ComObjCreate, ComObjGet, ComObjConnect, ComObjError, ComObjFlags, ObjAddRef/ObjRelease, ComObjQuery, ComObject , GetActiveObject (MSDN)

Examples

; Example (COM parameter): Pass a VARIANT ByRef to a COM function.
code := "
(
 Sub Example(Var)
 MsgBox Var
 Var = "out value!"
 End Sub
)"
sc := ComObjCreate("ScriptControl"), sc.Language := "VBScript", sc.AddCode(code)

var := ComVar()
var[] := "in value"
sc.Run("Example", var.ref)
MsgBox % var[]
