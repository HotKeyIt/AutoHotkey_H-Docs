Deref [v2]

Expands variable references and escape sequences contained inside other variables.

OutPutVar := Deref(String)
Example: value := Deref("The time now is %A_Hour%:%A_Min%:%A_Sec%")
Deref, OutPutVar, String 
Example: Deref, value,% "The time now is %A_Hour%:%A_Min%:%A_Sec%"

Parameters

OutputVar

The name of the variable in which to store the result.

String

A string containing variable references and/or escape sequences.

Remarks

Any badly formatted variable references will be omitted from the expanded result. The same is true if OutputVar is expanded into itself; in other words, any references to OutputVar inside String's variables will be omitted from the expansion (note however that String itself can be %OutputVar%).

Within a function, each variable reference in String may resolve to a local or global variable. Standard variable scope rules apply.

Examples

var1 := "test"
var2 := Chr(37) "var1"  ; Chr(37) is %
Deref OutputVar, %var2%
MsgBox %OutputVar%  ; Shows "test"
