GetTokenValue

Build in function to retrieve a value from an ExprTokenType structure.

OutputVar := GetTokenValue(TokenPointer)
Function Example: value := GetTokenValue(lpExprTokenType)

Parameters

lpExprTokenType

A pointer to an ExprTokenType structure.
This must not be a pointer to variable, string, floating-point number or numeric literal.

Return Value

Value from an ExprTokenType structure.
