IsBOM

Returns true if given address contains a byte order mark.


IsBOM, OutPutVar, Address [, BOM]
Command  Example: IsBOM, result,% &var, UTF-16BE
Function Example: result := IsBOM(&var, "UTF-16BE")

Parameters

OutPutVar

The name of the variable to store true if a byte order mark was found at given address.

Address

Memory address to check for a byte order mark.

BOM (optional)

Byte order mark to check for, default is UTF-8. This paramter can be UTF-8, UTF-16, UTF-16BE, UTF-32 or UTF-32BE only.

Examples

FileRead,script,*c %A_ScriptFullPath%
MsgBox % IsBOM(&script) ; Check if script is saved with UTF-8 BOM
