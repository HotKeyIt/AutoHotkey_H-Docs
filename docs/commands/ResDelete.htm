ResDelete

Deletes a resource from executable file (dll or exe).

ResGet, OutPutVar, Data, Name [, Type, Language]

Example:Success := ResDelete(A_AhkPath, "MYRES.RES")
Example: ResDelete, Success, %A_AhkPath%, MYRES.RES

Parameters

OutPutVar

The name of the variable in which to store the result.

Executable

Full path to executable file (dll or exe).

Name

The name of resource.

Type (optional)

Type of resource. Default is RCDATA (10).
See MSDN for default resource types.

Language

Resource language, default 1033 (en-us).
