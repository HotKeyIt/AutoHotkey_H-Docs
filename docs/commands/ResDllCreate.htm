ResDllCreate

Creates a resource only dll. Such dll will not have any executable code but can be loaded into programms to access the resouces.

ResDllCreate(DllPath)

Example:ResDllCreate(A_ScriptDir "\MyDll.dll")

ResDllCreate, DllPath

Example: ResDllCreate, %A_ScriptDir%\MyDll.dll

Parameters

DllPath

Path for created dll.

ErrorLevel

ErrorLevel will be set to 1 (true) if the file could not be created, otherwise it will be 0 (false).

Related

ResGet, ResGetEx, ResPut, ResFile, ResFiles, ResExist, ResDelete, ResDll, UnZipRawMemory

Example

ResDllCreate, %A_ScriptDir%\MyDll.dll
If ErrorLevel
	MsgBox The dll could not be created.
