ResExist

Check whether resource exists in the specified executable (dll or exe).

ResExist, OutPutVar, Name [, Type]

Command  Example: ResExist, IsScriptCompiled, %A_AhkPath%, >AHK_SCRIPT<
Function Example: IsScriptCompiled := ResExist(A_AhkPath, ">AHK_SCRIPT<")

Parameters

OutPutVar

The name of the variable in which to store 1 (true) if the resource exist or 0 (false) otherwise.

Executable

Full path to executable file (dll or exe).

Name

The name of resource.

Type (optional)

Type of resource. Default is RCDATA (10).
See MSDN for default resource types.

Related

ResGet, ResGetEx, ResPut, ResFile, ResFiles, ResExist, ResDelete, ResDll, UnZipRawMemory

Example

If !ResExist(A_AhkPath,"RESEXIST.AHK","LIB")
  MsgBox ResExist.ahk is not included in resource library.
