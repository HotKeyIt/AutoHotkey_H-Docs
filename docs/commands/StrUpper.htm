StrUpper

OutputVar := StrUpper(String , T)
Function Example: uppr := StrUpper("hello")

Parameters

OutputVar

The name of the variable in which to store the newly converted string.

String

The string to convert.

T

If this parameter is the letter T, the string will be converted to title case. For example, "GONE with the WIND" would become "Gone With The Wind".

Remarks

To detect whether a character or string is entirely uppercase or lowercase, use RegExMatch. For example:

var := "abc"
if RegExMatch(var, "^[a-z]$")
    MsgBox var is entirely composed of lowercase characters.
if !RegExMatch(var, "[A-Z]")
    MsgBox var does not contain any uppercase characters.
