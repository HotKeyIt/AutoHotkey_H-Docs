UnZipRawMemory

This function is used to unzip and decrypt raw memory, for example from resource.

UnZipRawMemory, OutPutVar, Address, Size, [, VariableOrAddress, Password]

Command  Example: UnZipRawMemory, Size,% &resource,% SizeOfResource(NULL,hRes), %Data%, AutoHotkey
Function Example: Size := UnZipRawMemory(&resource, SizeOfRes(NULL,hRes), Data, "AutoHotkey")

Parameters

OutPutVar

The name of the variable in which to store the size of unzipped memory in bytes.

Address

Address to raw zipped memory.

Size

Size of zipped memory.

VariableOrAddress (optional)

The name of a variable or address to copy unzipped memory to.

Password (optional)

The password used to encrypt the data.

Remarks

Ahk2Exe uses LiteZip to compress resources in raw mode (only 1 file supported).
Internally AutoHotkey is able to decompress raw resources automatically (compiled script + lib resources). This function can be used in script to decrypt and decompress raw resources.

Examples

ResourceRead(data,A_AhkExe,"BETWEEN.AHK","LIB")
UnZipRawMemory(&data,var,"AutoHotkey")
MsgBox % StrGet(&var,"UTF-8")
