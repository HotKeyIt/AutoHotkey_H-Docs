WinApiDef

Get the definition of a WinApi function.

WinApiDef, OutputVar, FunctionName

Command  Example: WinApiDef, Definition, LoadLibrary
Function Example: Definition := WinApiDef("LoadLibrary")

Parameters

OutputVar

WinApiDef returs the definition of a function that can be included automatically by WinApi, otherwise OutputVar is made blank.

FunctionName

The name of a function included in WinApi.

Related

DllCall, DynaCall, #DllImport, WinApi

Examples

MsgBox % WinApiDef("LoadLibrary")
