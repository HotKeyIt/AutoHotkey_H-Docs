addScript

Exported function that adds new code from string to currently executed script. 


addScript, OutPutVar, NewCode, WaitExecute

Command  Example: DllCall, OutPutVar, AutoHotkey.dll\addScript, Str, %NewCode%, Int, %WaitExecute%, UPTR
                  DllCall, OutPutVar, AutoHotkey.exe\addScript, Str, %NewCode%, Int, %WaitExecute%, UPTR
Function Example: OutPutVar := DllCall("AutoHotkey.dll\addScript", "Str", NewCode, "Int", WaitExecute, "UPTR")
                  OutPutVar := DllCall("AutoHotkey.exe\addScript", "Str", NewCode, "Int", WaitExecute, "UPTR")

Parameters

OutPutVar

The name of the variable in which to store the line pointer to new code, if code could not be added 0 will be stored.

NewCode

New Code to add to currently running script.

WaitExecute

0 = add code but do not execute.
1 = add code, execute and wait for return.
2 = add code, execute and return immediately (don't wait for code to end execution).

Related

ahkFindFunc, ahkFindLabel, ahkassign, DllCall()

Examples

dllpath:=A_ScriptDir "\AutoHotkey.dll"
DllCall("LoadLibrary","Str",dllpath) ; Load the AutoHotkey module.
DllCall(dllpath "\ahktextdll","Str","","Str","") ; start a new thread.
DllCall(dllpath "\addScript","Str","MsgBox Hello World!","Int",1) ; add and execute code

addScript("MsgBox Test",1) ; call addScript in current thread.
