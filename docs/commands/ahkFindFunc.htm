ahkFindFunc

Exported function that finds a function in currently executed script and returns a pointer to it. 


ahkFindFunc, OutPutVar, FuncName

Command  Example: DllCall, OutPutVar, AutoHotkey.dll\ahkFindFunc, Str, FuncName, PTR
                  DllCall, OutPutVar, AutoHotkey.exe\ahkFindFunc, Str, FuncName, PTR

Function Example: OutPutVar := DllCall("AutoHotkey.dll\ahkFindFunc", "Str", "FuncName", "PTR")
                  OutPutVar := DllCall("AutoHotkey.exe\ahkFindFunc", "Str", "FuncName", "PTR")

Parameters

OutPutVar

The name of the variable to store the function pointer in or 0 if the function was not found.

FuncName

The name of the function to find.

Related

ahkFunction, ahkPostFunction, ahkFindLabel, DllCall()

Examples

dllpath:=A_ScriptDir "\AutoHotkey.dll"
DllCall("LoadLibrary","Str",dllpath) ; Load the AutoHotkey module.
DllCall(dllpath "\ahktextdll","Str","~VKFF::Return`nMyFunc(param){`nReturn param`n}","Str","") ; start a new thread, just the function.
MsgBox % DllCall(dllpath "\ahkFindFunc","Str","MyFunc","PTR") ; call the function and display returned value in MsgBox.ahkFindFunc("MyFunc") ; find function in current thread
