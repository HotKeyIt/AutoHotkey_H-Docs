ahkFindLabel

Exported function that finds a label in currently executed script and returns a pointer to it. 


ahkFindLabel, OutPutVar, LabelName

Command  Example: DllCall, OutPutVar, AutoHotkey.dll\ahkFindLabel, Str, LabelName, PTR
                  DllCall, OutPutVar, AutoHotkey.exe\ahkFindLabel, Str, LabelName, PTR

Function Example: OutPutVar := DllCall("AutoHotkey.dll\ahkFindLabel", "Str", "LabelName", "PTR")
                  OutPutVar := DllCall("AutoHotkey.exe\ahkFindLabel", "Str", "LabelName", "PTR")

Parameters

OutPutVar

The name of the variable to store the label pointer in or 0 if the label was not found.

LabelName

Name of the label to find.

Related

ahkFunction, ahkPostFunction, ahkFindLabel, DllCall()

Examples

dllpath:=A_ScriptDir "\AutoHotkey.dll"
DllCall("LoadLibrary","Str",dllpath) ; Load the AutoHotkey module.
DllCall(dllpath "\ahktextdll","Str","~VKFF::Return`nMyLabel:`nReturn","Str","") ; start a new thread, just the function.
MsgBox % DllCall(dllpath "\ahkFindFunc","Str","MyLabel","PTR") ; call the function and display returned value in MsgBox.ahkFindLabel("MyLabel") ; find label in current thread
