ahkLabel

Exported function that allows calling a function in script currently executed by AutoHotkey module.
Paramenters can be strings only as well as return value.


DllCall, OutPutVar, Module\ahkLabel, Str, LabelName, UInt, DoNotWait, CDECL CHAR

Command  Example: DllCall, OutPutVar, AutoHotkey.dll\ahkLabel, Str, MyLabel, UInt, 0, CDECL CHAR
                  DllCall, OutPutVar, AutoHotkey.exe\ahkLabel, Str, MyLabel, Uint, 0, CHAR

Function Example: OutPutVar := DllCall("AutoHotkey.dll\ahkLabel", "Str", "MyLabel", "UInt", 0, "CDECL CHAR")
                  OutPutVar := DllCall("AutoHotkey.exe\ahkLabel", "Str", "MyLabel", "UInt", 0, "CDECL CHAR")

Parameters

OutPutVar

The name of the variable to store 1 / true if label was found or 0 / false otherwise.

LabelName

The name of the label to jump to.

DoNotWait

1 (true) will not wait for the code to finish / return, FALSE / NULL / 0 will wait for execution to finish like GoSub does.

Related

ahkPostFunction, ahkFindFunc, DllCall()

Examples

dllpath:=A_ScriptDir "\AutoHotkey.dll"
DllCall("LoadLibrary","Str",dllpath) ; Load the AutoHotkey module.
DllCall(dllpath "\ahktextdll","Str","
(
~VKFF::Return
MyLabel:
MsgBox `% A_ThisLabel
Return
)","Str","") ; start a new thread, just the function.
DllCall(dllpath "\ahkLabel","Str","MyLabel","UInt",0) ; jump to label and wait for it to finish / return.
