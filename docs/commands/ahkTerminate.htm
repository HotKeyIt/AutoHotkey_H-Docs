ahkTerminate

Exported function that exits script currently executed by AutoHotkey module.
This function is only available in AutoHotkey[Mini].dll 


DllCall,, AutoHotkey.dll\ahkTerminate, Int, Timeout, CHAR

Command  Example: DllCall,, AutoHotkey.dll\ahkTerminate, Int, 500, CHAR

Function Example: DllCall("AutoHotkey.dll\ahkTerminate", "Int", 500, "CHAR")

Parameters

Timeout

Time to wait until thread exits, use FALSE / NULL / 0 = no Timeout. NOTE: using a negative vaue you can force exit after given Timeout, with a positive Timeout it might take longer to exit thread but assures a clean exit.

Related

ahkPostFunction, ahkFindFunc, DllCall()

Examples

dllpath:=A_ScriptDir "\AutoHotkey.dll"
DllCall("LoadLibrary","Str",dllpath) ; Load the AutoHotkey module.
DllCall(dllpath "\ahktextdll","Str","~VKFF::Return`nMyFunc(param){`nSleep 10000`nMsgBox `% param`n}","Str","") ; start a new thread, just the function.
DllCall(dllpath "\ahkFunction","Str","MyFunc","Str","Hello World!","Str") ; call the function.
