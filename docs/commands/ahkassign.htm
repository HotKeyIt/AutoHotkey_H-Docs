ahkassign

Exported function that assins a string value to a variable in currently executed script. 


ahkassign, OutPutVar, VarName, Value

Command  Example: DllCall, OutPutVar, AutoHotkey.dll\ahkassign, Str, Variable, Str, 100, UInt
                  DllCall, OutPutVar, AutoHotkey.exe\ahkassign, Str, Variable, Str, 100, Int

Function Example: OutPutVar := DllCall("AutoHotkey.dll\ahkassign", "Str", "Variable", "Str", "100" ,"Int")
                  OutPutVar := DllCall("AutoHotkey.exe\ahkassign", "Str", "Variable", "Str", "100", "UInt")

Parameters

OutPutVar

The name of the variable in which to store the result, -1 on failure and 0 on success.

VarName

Name of the variable to assign value to.

Value

Value as String.

Related

ahkFindFunc, ahkFindLabel, ahkassign, DllCall()

Examples

dllpath:=A_ScriptDir "\AutoHotkey.dll"
DllCall("LoadLibrary","Str",dllpath) ; Load the AutoHotkey module.
DllCall(dllpath "\ahktextdll","Str","","Str","") ; start a new thread, just the function.
DllCall(dllpath "\ahkassign","Str","MyVar","Str","Hello World!") ; Assign value to variable
MsgBox % DllCall(dllpath "\ahkgetvar","Str","MyVar","UInt",0,"STR") ; Retrieve value from variable
