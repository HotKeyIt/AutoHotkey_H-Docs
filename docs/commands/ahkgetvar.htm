ahkgetvar

Built in function to retrieve the value of a variable as string.


DllCall, OutPutVar, Module\ahkgetvar, Str, VarName, UInt, GetVarPointer, CDECL UInt

Command  Example: DllCall, OutPutVar, AutoHotkey.dll\ahkgetvar, Str, MyVar, UInt, 0, CDECL UInt
                  DllCall, OutPutVar, AutoHotkey.exe\ahkgetvar, Str, MyVar, UInt, 0, UInt

Function Example: OutPutVar := DllCall("AutoHotkey.dll\ahkgetvar", "Str", "MyVar", "Str", 0 ,"CDECL UInt")
                  OutPutVar := DllCall("AutoHotkey.exe\ahkgetvar", "Str", "MyVar", "Str", 0, "CDECL UInt")

Parameters

OutPutVar

The name of variable in which to store the value of variable as string or variable pointer if GetvarPointer is 1 / true.

VarName

The name of the variable to get pointer or contents from.

GetVarPointer

1 / true to receive a pointer to variable, FALSE / NULL / 0 to receive content of variable.

Related

Struct, DllCall()

Examples

dllpath:=A_ScriptDir "\AutoHotkey.dll"
DllCall("LoadLibrary","Str",dllpath) ; Load the AutoHotkey module.
DllCall(dllpath "\ahktextdll","Str","","Str","") ; start a new thread, just the function.
DllCall(dllpath "\ahkassign","Str","MyVar","Str","Hello World!") ; Assign value to variable
MsgBox % DllCall(dllpath "\ahkgetvar","Str","MyVar","UInt",0,"STR") ; Retrieve value from variable
