Ahk2Exe

Custom Ahk2Exe for AutoHotkey_H.
Copy AutoHotkey.exe to compiler folder and rename it to Ahk2Exe.exe to run the compiler.

The compiler included in downloads supports additional AutoHotkey_H features.

Resources can be compressed to reduce size of the program.

You can compile scripts with AutoHotkey.exe or AutoHotkey.dll.

Main Script can be encrypted for protection, but you should recompile AutoHotkey_H in Visual Studio with a different password to have proper protection.

WinApi functions are automatically declared for AutoHotkeySC.bin in compiled script using #DllImport.
