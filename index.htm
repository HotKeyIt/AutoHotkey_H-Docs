AutoHotkey_H

AutoHotkey_H adds functionality to original AutoHotkey and offers true multi-threading using AutoHotkey.dll.

Downloads

These downloads include AutoHotkey.exe and AutoHotkey.dll, Documentation and some miscellaneous function scripts.

  • AutoHotkey_H v1 - download v1 zip package.
  • AutoHotkey_H v2 alpha - download v2 alpha zip package.
  • Source Code - download via git or click the "ZIP" button at github.

Microsoft Visual C++ 2017 has been used to build these executables. Windows XP and later are supported.
