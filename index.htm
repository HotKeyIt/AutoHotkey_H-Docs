AutoHotkey_H

AutoHotkey_H adds functionality to original AutoHotkey and offers true multi-threading using NewThread() function or AutoHotkey.dll.

Downloads

These downloads include AutoHotkey.exe and AutoHotkey.dll, Documentation and some miscellaneous function scripts.

Note, AutoHotkey_H v1.1 documentation is out of date. Please refer to AutoHotkey v2 documentation, majority of functions are included in v1.1.

  • AutoHotkey_H v1 - download v1 zip package.
  • AutoHotkey_H v2 alpha - download v2 alpha zip package.
  • Source Code - download via git or click the "ZIP" button at github.

Microsoft Visual C++ 2013 has been used to build these executables. Windows XP and later are supported.

Source code is available at github.
